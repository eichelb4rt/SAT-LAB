usage: random-cnf.py [-h] [-o output_dir] t n c k

positional arguments:
  t                     Number of CNFs
  n                     Number of variables
  c                     Number of clauses
  k                     Clause width

optional arguments:
  -h, --help            show this help message and exit
  -o output_dir, --output output_dir
                        Directory that the CNFs are written to
